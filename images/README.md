# Images Required

Before launching, add the following images to this folder:

## Required

- **logo.png** - Main logo (copy from BirdIcon.png in the main project, resize to ~200x200)
- **favicon.png** - Browser favicon (32x32 or 64x64)

## Screenshots (for gallery section)

Add these to a `screenshots/` subfolder:

- **main-app.png** - Main application window showing live feed and detection history
- **detection-snapshot.png** - Example of a bird detection with overlay
- **notification-example.png** - Screenshot of a push notification or email
- **statistics-view.png** - Statistics/reports view

## Recommended Image Specs

| Image | Dimensions | Format |
|-------|------------|--------|
| logo.png | 200x200 | PNG (transparent) |
| favicon.png | 64x64 | PNG |
| Hero screenshot | 800x600 | PNG or JPG |
| Gallery screenshots | 600x400+ | PNG or JPG |

## Quick Setup

1. Copy `../BirdIcon.png` to `logo.png`
2. Create a 64x64 version as `favicon.png`
3. Take app screenshots and save to `screenshots/`
4. Update `index.html` to reference the real images instead of placeholders
